# Task4java

##  Overview

This is a simple console-based Notes Manager application built using **Java FileReader, FileWriter, and BufferedReader**.  
It allows users to:

- Add notes (saved permanently in a file)
- View previously saved notes
- Understand basic Java File I/O operations

## Technologies Used

- Java
- VS Code / IntelliJ / Any IDE
- FileReader, FileWriter, BufferedReader
